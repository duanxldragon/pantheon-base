### File 1: `backend/modules/system/config/setting/setting_handler.go`

In method `ServeUploadedFile`, the call `http.ServeFileFS(c.Writer, c.Request, os.DirFS(rootPath), objectKey)` uses `objectKey` derived from user input (`c.Param("filepath")` via `NormalizeObjectKey`). Add a `filepath.IsLocal()` guard on `objectKey` right before the `http.ServeFileFS` call on line ~209. If `IsLocal` returns false, return an upload-file-not-found error. Example pattern:

```go
if !filepath.IsLocal(objectKey) {
    common.Fail(c, common.CodeParamInvalid, "upload.file.not_found")
    return
}
http.ServeFileFS(c.Writer, c.Request, os.DirFS(rootPath), objectKey)
```

### File 2: `backend/modules/system/dynamicmodule/dynamic_module_registry.go`

Even though `generatedModuleRelativePath` sanitizes them, add an inline `filepath.IsLocal()` check on `relativeTarget` just before `os.ReadFile(target)` as defense-in-depth (static analysis cannot trace validation across function boundaries). Example:

```go
if !filepath.IsLocal(relativeTarget) {

```
