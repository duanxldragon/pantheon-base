# Platform Shell Dual-Mode Acceptance Template

Chinese version: [PLATFORM_SHELL_DUAL_MODE_ACCEPTANCE_TEMPLATE.md](./PLATFORM_SHELL_DUAL_MODE_ACCEPTANCE_TEMPLATE.md)

This is the fixed acceptance template for `platform` shell changes.

It is intended for:

- shell-related PRs
- staged platform records
- shell convergence summaries

## Applies To

- `frontend/src/core/layout/index.tsx`
- `frontend/src/core/layout/index.css`
- dynamic-menu rendering
- menu icon mapping
- top horizontal navigation
- left vertical navigation
- adjacent brand, tab, and top-bar layout

## Required Inputs

The template expects explicit checks for:

- expanded vertical sidebar
- collapsed vertical sidebar
- horizontal navigation
- horizontal submenu popups

Use the Chinese source document for the full template body and field-by-field acceptance form.
