# Platform UI Migration Acceptance Matrix (2026-04-30)

Chinese version: [PLATFORM_ACCEPTANCE_MATRIX_20260430_UI_MIGRATION.md](./PLATFORM_ACCEPTANCE_MATRIX_20260430_UI_MIGRATION.md)

This matrix captures the platform-layer UI migration baseline established on April 30, 2026.

It answers a practical question: which shell and system pages have already entered the new shared skeleton, which still mix old patterns, and which areas remain visually pending.

## Status Vocabulary

- `Target`: acceptable shared platform direction
- `Mixed`: partly migrated, not safe to copy
- `Legacy`: known historical residue
- `Pending`: implementation exists but final visual strategy is not fully closed

## Main Focus Areas

- platform shell and navigation strategy
- right-rail convergence
- modal and drawer convergence
- reusable layout anchors
- pages that are safe or unsafe to copy forward
